# Oma Prototyyppi - Löydä oma yhteisösi

Tämä on verkkosivuston prototyyppi, joka on toteutettu HTML/CSS/JS:llä käyttäen mobile-first -lähestymistapaa.

## Sivut

- **index.html**: Etusivu
- **categories.html**: Kategoria-näkymä
- **organization.html**: Yhteisön esittelysivu (malli)
- **event.html**: Tapahtuman esittelysivu (malli)
- **networking.html**: Mitä verkostoituminen tarkoittaa -sivu

## Käytetyt värit ja fontit

- Värit: Brandbookin mukaan (#69013b, #fce0d1, #ffffff, #191919)
- Fontit: Playfair Display (otsikot), Lato (teksti)

## Kieliversiot

Sivut tukevat suomea ja englantia. Kielenvaihto oikeassa yläkulmassa.

## Navigaatio

Ylävalikko linkkeineen.

## Käynnistäminen

Avaa index.html selaimessa tai käynnistä paikallinen palvelin:


Sitten avaa http://localhost:8000/index.html

Prototyyppi perustuu workshopin ideoihin: yhteisöjen ja tapahtumien löytäminen, kategorisointi, suodatus, kieliversiot.
